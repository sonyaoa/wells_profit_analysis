# Проект: Анализ прибыльности регионов добычи нефти

## Описание
Проект посвящён оценке трёх регионов добычи нефти с целью выбора наиболее прибыльного и наименее рискованного для инвестиций.

## Цель исследования
Определить регион для разработки новых скважин, основываясь на:
- прогнозировании объёмов добычи;
- расчёте прибыли с учётом бюджета и цены ресурса;
- оценке рисков убытков.

## Ход исследования
1. Предобработка данных: обработка пропусков, кодирование категориальных признаков, подготовка данных к анализу.
2. Исследовательский анализ данных (EDA): выявление закономерностей, корреляций и трендов.
3. Обучение моделей прогнозирования объёма запасов.
4. Оценка качества моделей по метрикам (RMSE, среднее предсказанное значение).
5. Расчёт прибыли и рисков методом bootstrap.
6. Итоговые выводы и рекомендации для заказчика.

## Технологии
Python, Pandas, NumPy, Matplotlib, Scikit-learn, Bootstrap, Визуализация данных

## Ключевые результаты
- Определен наиболее перспективный регион нефтедобычи.
- Средняя прибыль: ≈441 млн руб.
- Риск убытков: 1.6% (ниже установленного порога).
- Сформированы рекомендации по выбору региона для инвестиций.
